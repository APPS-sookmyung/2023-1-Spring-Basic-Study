# 4주차 과제

### CRUD

: 대부분의 컴퓨터 소프트웨어가 가지는 기본적인 데이터 처리 기능 (Create, Read, Update, Delete)

| 이름 | 조작 | SQL |
| --- | --- | --- |
| Create | 생성 | INSERT |
| Read | 조회 | SELECT |
| Update | 수정 | UPDATE |
| Delete | 삭제 | DELETE |

Create: 데이터 생성

Read: 데이터 읽기

Update: 새로운 데이터 업데이트

데이터베이스 또는 API와 같은 스토리지 시스템의 기존 데이터를 수정, 사용자는 응용 프로그램 내에서 이전에 저장된 정보 변경 가

Delete: 데이터 삭제

데이터베이스 또는 API와 같은 스토리지 시스템의 기존 데이터를 제거, 사용자는 특정 레코드나 리소스를 영구적으로 삭제 가능
